# SEOAnalyser
Build on .NET Core 2.0, Angular

1. use browser to browse :http://localhost:14244/home

2. input Text or URL inside the text box and select any necessery option: filters out stop-words, number of occurrences on the page of each word, number of occurrences on the page of each word listed in meta tags, number of external links in the text 

3. Click on "Search" button

4. result will be shown by clicking the Tab : Word Occurance / Meta Tag / External Link.
    * result shown is based on the option 2a-d.
